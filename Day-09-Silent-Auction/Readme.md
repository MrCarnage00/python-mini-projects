# Day 09 - Silent Auction

A command-line silent auction application built in Python.

## Features

- Accepts multiple bids
- Stores bidder information
- Finds the highest bidder
- Displays the winner

## Concepts Practiced

- Functions
- Dictionaries
- Loops
- Conditional Statements
- User Input

## Files

- `main.py` - Contains the auction logic.
- `art.py` - Stores the ASCII art/logo used in the program.

## Run

```bash
python main.py
